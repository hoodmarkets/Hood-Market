# Audit Reports

These audit reports were conducted on the **original Clanker v4 contracts** ([clanker-devco/v4-contracts](https://github.com/clanker-devco/v4-contracts)) prior to this fork. They are included here for reference only.

**These audits do not cover Liquid Protocol's modifications**, including the Clanker-to-Liquid rebrand, removal of deprecated v4.0 contracts, or any future changes made in this repository.

Liquid Protocol has not independently commissioned an audit of this codebase.
