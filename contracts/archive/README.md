# Archive

Files preserved here were removed from active paths during a repository cleanup. They are kept for reference and historical record.

## Contents

| Directory | Description |
|-----------|-------------|
| `dev-artifacts/` | Files generated by AI coding tools during development (e.g. `CLAUDE.md`) |
| `stale-abis/` | ABIs for contracts that have been renamed or removed — superseded by current `base_mainnet_abis/` entries |
| `stale-deployments/` | Intermediate and failed Foundry broadcast runs from the initial mainnet deployment — superseded by the final `broadcast/` entries |
